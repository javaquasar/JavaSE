Описать аннотацию Init для описания некоторого класса-сервиса.
 Свойство toInit логического типа должно показывать, запускать 
 ли автоматически метод, помеченный аннотацией Init.

Разместить классы с соответствующими аннотациями в различных 
подкаталогах некоторого каталога. Создать программу обхода 
всех подкаталогов, загрузки классов и выполнения функций, 
помеченных аннотацией Init, если у аннотации перед классом 
установлено свойство toInit. 

Для загрузки классов использовать URLClassLoader. Запустить 
поиск классов в отдельном потоке управления с возможностью 
прервать его из главного потока.
